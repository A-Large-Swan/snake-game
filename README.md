# Snake Game

这是一个在Linux终端运行的贪吃蛇游戏。

## 项目介绍

这是一个使用C语言编写的经典贪吃蛇游戏，在Linux终端中运行。游戏具有简单的操作方式和清晰的界面显示，让玩家可以享受到经典贪吃蛇游戏的乐趣。

## 安装说明

### 环境要求
- Linux操作系统
- GCC编译器
- Make工具

### 安装步骤

1. 克隆项目到本地：
```bash
git clone https://github.com/A-Large-Swan/snake-game.git
```

2. 进入项目目录：
```bash
cd snake-game
```

3. 执行安装脚本：
```bash
sudo sh install.sh
```

## 使用方法

1. 在项目目录下运行游戏：
```bash
./snake
```

2. 游戏开始后，使用方向键↑、↓、←、→控制蛇的移动方向。

## 游戏规则

1. 控制蛇吃掉随机出现的食物
2. 每吃掉一个食物，蛇的长度增加
3. 撞到墙壁或自己的身体游戏结束
4. 分数随着吃掉的食物数量增加

## 技术栈

- 编程语言：C
- 编译工具：GCC
- 构建工具：Make
- 运行环境：Linux终端
